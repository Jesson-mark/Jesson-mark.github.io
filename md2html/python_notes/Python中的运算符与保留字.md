# 算术运算符
- +, -, *, / 分别是加、减、乘、除
- //：整除，3//2=1, 3.0//2=1.0
- *\*：幂，2**4=16
- %：整除求余数，10%3=1

# 比较运算符
`返回值是bool类型`

a==b：判断ab是否相等
a!=b：判断ab是否不等
\>：大于
<：小于
\>=：大于等于
<=：小于等于

# 成员运算符
`返回值是bool类型`
in
not in

# 逻辑运算符
and, or, not

# 身份运算符
`返回值是bool类型`
is, not is
